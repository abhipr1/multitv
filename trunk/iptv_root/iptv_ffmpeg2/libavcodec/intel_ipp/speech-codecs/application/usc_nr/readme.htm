Overview

The USC Noise reduction sample is a command line based application. It supports two components: Noise Reduction and Voice Activity Detection.

The Noise reduction sample’s component processes input PCM file narrow band 16bit 8000 Hz and creates output PCM file narrow band 16bit 8000 Hz.

The VAD detection sample’s component processes input PCM file narrow band 16bit 8000 Hz or wide band 16bit 16000 Hz and creates output PCM file narrow band 16bit 8000 Hz or wide band 16bit 16000 Hz respectively.

Running the Software

For information on building the sample, see the readme file in the ipp_samples\speech-codecs directory.

In order to run the executable created by this sample,

be sure that the LD_LIBRARY_PATH environment variable includes the directories of the .so files from the Intel ® IPP installation. This can be accomplished by entering the following at the command prompt:   for examples 

export LD_LIBRARY_PATH=/opt/intel/ipp/5.3/ia32/sharedlib:$LD_LIBRARY_PATH

or by running the shell script ippvars32.sh from the Intel ® IPP installation (located in /opt/intel/ipp/5.3/ia32/tools/env/ by default).

./usc_nr [options] <inPCMFile> <outPCMFile> 

The option list:  

-alg IPP_NR_EPHM|IPP_NR_EC|IPP_VADG723|IPP_VADG729_FP|IPP_VADG729|IPP_VAD - signal filter algorithm name.

IPP_NR_EPHM - Ehprame-Malah based Noise Reduction algorithm.

IPP_NR_EC - IPP based Noise Reduction algorithm.

IPP_VADG723 - ITU G.723.1 VAD algorithm.

IPP_VADG729_FP - ITU G.729 floating-point VAD algorithm.

IPP_VADG729 - ITU G.729 VAD algorithm.

IPP_VAD - IPP based VAD algorithm.

-f<freq>              Sample frequency in Hertz. 16000 for IPP_VAD algorithm, 8000 for others.

-n                        output additional information  (optional).

-list                     Enumerate linked USC signal filter algorithms and exit  (optional).

  -o <logFile>      write to <logFile> file log information (optional).
<inPCMFile> - input audio PCM file 8000 Hz (IPP_NR_EPHM, IPP_NR_EC, IPP_VADG723, IPP_VADG729_FP, IPP_VADG729 algorithms) or 16000 Hz (IPP_VAD algorithm).

<outPCMFile> - output audio PCM file 8000 Hz (IPP_NR_EPHM, IPP_NR_EC, IPP_VADG723, IPP_VADG729_FP, IPP_VADG729 algorithms) or 16000 Hz (IPP_VAD algorithm).
