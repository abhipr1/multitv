README IP.TV 0.2.3 BETA - Windows
=================================

RUNNING
-------
Before running iptv_client.exe, please install the following packages:

1. OpenAL Installer (oalinst.exe)
2. Windows Installer 3.1 (WindowsInstaller-KB893803-v2-x86.exe)
3. Visual Studio 2005 Redistributable (vcredist_x86.exe)

Run iptv_client.exe.

Note
----
The package "Windows Installer 3.1" is a dependency of "Visual Studio 2005 Redistributable".
They should be installed in the indicated order.
