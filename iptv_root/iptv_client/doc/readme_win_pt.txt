README IP.TV 0.2.3 BETA - Windows
=================================

Executando o client IP.TV
-------------------------
Antes de executar o programa iptv_client.exe, instale os seguintes pacotes:

1. OpenAL Installer (oalinst.exe)
2. Windows Installer 3.1 (WindowsInstaller-KB893803-v2-x86.exe)
3. Visual Studio 2005 Redistributable (vcredist_x86.exe)

Execute o arquivo iptv_client.exe.

Observação
----------
O pacote "Windows Installer 3.1" é uma dependência de "Visual Studio 2005 Redistributable".
Eles devem ser instaldos na ordem indicada.
